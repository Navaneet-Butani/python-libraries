This repository includes information about Python's requests library.

This contains 2 files, 
1) requests_demo1.py : Which contains some common and basic information regarding to the 'requests' library.
2) requests_demo2.py : Which contains 4 types of request(GET, POST, AUTH and Dynamic data) information and procedure of it.

First of all you have install that library in your system with 'pip install requests' command.
Then follow line and part by part into the files.
Do commenting and uncommenting as per requirement.
