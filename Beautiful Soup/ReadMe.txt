Beautiful Soup is a Python package for parsing HTML and XML documents. 
It creates a parse tree for parsed pages that can be used to extract data from HTML, which is useful for web scraping.

Here in this repository contains three python files and one html content file.
Go as per numbers
1.bs_and_requests_basics.py : Which includes basics of Requests and Beautiful Soup.
2.white_house_example.py : Which is a practical example with the implementation.
3.ba_objects.py : Which inculdes higher level that the basics.

For implementing this tutorial in your system you have to install some packages
1) Requests by 'pip install requests'
2) Beautiful Soup by 'pip install beautifulsoup4'
3) lxml by 'pip install lxml'

Notes : 
Keep your internet connection ON during this.
Follow these files line by line.
Do commenting and uncommenting as per requirements.
Which lines contains '##' at starting, are the coments.
Which lines contain '#' at starting are python commands.
